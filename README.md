# Stopwatch Web Application

A responsive and interactive Stopwatch Web Application built using HTML, CSS, and JavaScript.

## Features
- ▶️ Start Stopwatch
- ⏸️ Pause Stopwatch
- 🔄 Reset Stopwatch
- 🏁 Record Lap Times
- 📱 Responsive Design
- ⚡ Accurate Time Tracking

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Learning Outcomes
- DOM Manipulation
- Event Handling
- JavaScript Timers (`setInterval` and `clearInterval`)
- Responsive Web Design
